# AICO Frontend

AI 학습 코치를 목표로 하는 **AICO** 프로젝트의 프론트엔드입니다.  
React를 기반으로 학습 진척도 관리, 질의응답(GPT), 그룹 스터디 채팅 등 학습 보조 기능을 제공합니다.

---

## Default Folder Structure

aico-frontend/
├── public/
├── src/
│   ├── assets/         # 이미지, 아이콘 등
│   ├── components/     # 공통 UI 컴포넌트
│   ├── features/       # 기능(도메인)별 폴더
│   ├── hooks/          # 커스텀 훅
│   ├── pages/          # 라우팅되는 페이지 컴포넌트
│   │   ├── Dashboard.jsx
│   │   └── LoginPage.jsx
│   ├── router/         # 라우터 설정 (AppRoutes 등)
│   ├── styles/         # 전역 스타일 (App.css, index.css)
│   │   ├── App.css
│   │   └── index.css
│   ├── App.jsx
│   └── main.jsx
├── .gitignore
├── eslint.config.js
├── index.html
├── package-lock.json
├── package.json
├── README.md
└── vite.config.js
